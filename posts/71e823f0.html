从零开始配置nvim

Aidens-fox
2026-07-07
linux, 软件

前言

nvim(neovim),一个一款基于Vim的文本编辑器，你可以理解为lua版的vim，因为nvim可以使用lua语言来配置，支持lua让nvim的启动速度变得更快，所以现在主流都是选择nvim来作为文本编辑器,那么本篇博文我将会教你如何配置nvim和安装插件，如果你连vim怎么退出都不会我建议你先看看官方文档(-:

安装nvim

archlinux直接使用包管理pacman安装:
sudo pacman -S neovim

新建配置文件

nvim我建议是不要把所以配置都写在一个文件下，不然在时间一长绝对会乱死，在.config目录下新建nvim文件夹:
mkdir .config/nvim

在nvim目录下新建init.lua文件(类似于vimrc)
touch init.lua

在安装插件之前我们需要先配置一些nvim本身，所以先不创建lua文件夹，因为那是插件管理器和插件的文件夹，那么我们打开init.lua
为了方便所以把vim.opt(类似:set)局部变量opt
local opt = vim.opt

行号在敲代码的时候十分重要所以我们需要开启
开启绝对行号
opt.number = true
开启相对行号
opt.relativenumber = true

在敲代码时如果代码被包裹(折行)了，也就是在当前行没有显示的内容自动到下一行了，如果是阅读文章这没有什么关系，但如果是敲代码的这就会破坏结构，所以如果不是阅读文章一般是关闭的
opt.wrap = false

在使用编辑器的时候我们可以高亮光标(高亮当前行或列)来更快清楚当前行数
开启高亮显示当前行
opt.cursorcolumn = true
开启高亮显示当前列(两个一起形成十字架,来驱除报错(-:)
opt.cursorline = true

在搜索的时候，为了方便所以建议是关闭大小写敏感的
opt.ignorecase = true
opt.smartcase = true

在默认的nvim中的剪贴板是不同步系统剪贴板的，所以我们需要设置同步
opt.clipboard:append("unnamedplus")
